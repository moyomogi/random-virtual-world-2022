## ローカルで見る方法
WSL の上で作業してください。  
まず `Code > Download ZIP` から DL してカレントフォルダに置いてください。その後、下記コマンドを実行してください。  
```sh
npm install  # ./package.json を元に依存コマンドをインストール
npm run dev
```

## Firebase にデプロイ
```sh
firebase deploy
```
